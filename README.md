## Desenvolvimento WEB - Atividade 3 - Acesso ao BD

###Tema

 Criar uma aplicação web de cadastro, listagem e exclusão de usuários, com frontend estático em HTML/CSS/JavaScript e backend em Node.js com Express conectado ao PostgreSQL e deploy em ambiente de nuvem.

### Objetivo

- Compreender o funcionamento de requisições HTTP em aplicações web utilizando JavaScript no navegador.
- Utilizar a função fetch para consumir o backend e obter dados em formato JSON.
- Manipular o DOM (Document Object Model) para atualizar dinamicamente o conteúdo da página.
- Implementar validação básica de dados de entrada antes de realizar uma requisição à API.
- Exibir diferentes estados da interface (carregamento, erro e sucesso) durante o processo de consulta.
- Integrar front-end dinâmico com uma aplicação web servida por Node.js e Express.
